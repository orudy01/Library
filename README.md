# Library

From Odin Project

Welcome to Rodolfo Ortegas's Book Tracking Project, an assignment within the Odin Project curriculum. The goal of this repo is to practice the following skill sets:

- JavaScript
  - Classes
    - Constructors
  - Array Methods
    - `splice()`
  - DOM Manipulation
    
- CSS
  - Grid
  - Flex
